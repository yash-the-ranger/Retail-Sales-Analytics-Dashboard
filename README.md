# Retail Sales Analytics Dashboard

## Project Objective

The objective of this project is to analyze retail sales data and create an interactive dashboard to track key business metrics such as total sales, profit, quantity sold, category-wise performance, region-wise performance, and sales trends.

This dashboard helps understand business performance and supports better decision-making through clear visual insights.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts
* Slicers
* Data Cleaning
* Dashboard Design

## Dataset Description

The dataset contains retail sales transaction details including order information, product categories, sales amount, profit, quantity, region, and date-related fields.

## Key Metrics

* Total Sales
* Total Profit
* Total Quantity Sold
* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Top Performing Products

## Project Workflow

1. Collected and imported the retail sales dataset.
2. Cleaned the data by checking missing values and duplicate records.
3. Created Pivot Tables for sales, profit, category, and region analysis.
4. Built charts to visualize important business metrics.
5. Added slicers for interactive filtering.
6. Designed a dashboard layout with KPIs and visual insights.
7. Generated business insights from the dashboard.

## Dashboard Features

* KPI cards for sales, profit, and quantity
* Category-wise sales analysis
* Region-wise performance analysis
* Monthly sales trend visualization
* Product performance analysis
* Interactive slicers for better filtering

## Business Insights

* Identified the highest-performing product categories.
* Analyzed which regions generated the most sales.
* Tracked monthly sales trends to understand business growth.
* Compared product performance based on sales and profit.
* Created a dashboard that helps users quickly understand retail performance.
